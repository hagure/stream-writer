# Stream Writer 0.1

## What it does:
- Updates contents of specific text files to be used as Text Objects in streaming apps such as OBS & GameShow

## Requirements

- [Pashua](https://www.bluem.net/en/projects/pashua/)

## Setup

1. Create Necessary Text Files
2. Choose Directory

## Input

- Player Names
- Player Scores

## Output

- p1_name.txt
- p1_score.txt
- p2_name.txt
- p2_score.txt
